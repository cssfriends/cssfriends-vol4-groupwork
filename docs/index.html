CSSフレンズ #4「CSSで失敗した暴露話とリファクタリング」

会場： サイボウズ 大阪梅田オフィス

日程： 2018/03/04(日) 14:00 〜 17:00

発表について

座談会「主催者陣によるCSSの失敗話、つらかった話。そこからどう挽回したか」

司会： 榊原昌彦

パネラー： GOUTEN・kamino・Daisuke KONISHI・シマキョウスケ

今まで業務において、CSSでどのようにつらい思いをしてきたか。自分がやってしまった致命的な失敗、挽回可能な失敗をどう軌道修正したか。CSSのファイル・フォルダ構造はどうしてるか。赤裸々に恥ずかしい話が、今日ここで語られます！

グループワーク「ダメCSS設計の事例ディスカッション ーー これってどうしたらいいの？」

各テーブルにメンターがついて、CSSフレンズ主催者陣が考える「ダメCSS設計」を批判し、このまま突き進んだら後悔しかしない未来に涙し、どうやって変えていったらいいかをディスカッションしましょう。CSS設計に正解はありませんが、明らかにだめな事例はあるのです！

おやつ休憩について

皆様から持ち寄っていただいた（予定）お菓子を食べながら参加者同士で、仕事の悩みや勉強で困っているところ、またセッションでわからなかった！という内容を話題に、お菓子とドリンクを囲みながら懇親しましょう。
CSSについて熱く語るも良し、悩み相談するもよし。 Webの情報を交換しあう友だちをつくる機会にして下さい！

このイベントについて

毎回がmeet upイベントなので、「入りにくそう」「わからなかったらどうしよう」という心配はいりません。 CSSを学びたい、日頃の業務で困ってることを聞きたい、友だちをつくりたい、サイボウズのきれいなオフィスに遊びにいってみたいなど、何らかで興味をもった方はぜひお越し下さい！

今回が2017年度最終回です。2018年度はまた「CSSとは」からはじめます。
